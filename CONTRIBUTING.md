### Requirements

- [Node.js v20](https://nodejs.org/en)
- [Pnpm](https://pnpm.io/)
- [Foundry](https://book.getfoundry.sh/)

### Setup

> Installation is local and doesn't require root privileges.

Install dependencies

```bash
pnpm install
```

### Test

Run Hardhat tests

```bash
pnpm test
```

See `package.json` for more commands.
